# File Upload To Anki Workflow

When I upload a lecture, PDF, notes file, or screenshot:

1. Identify the topic.
2. Extract high-yield concepts.
3. Separate into:
   - Definitions
   - Mechanisms
   - Pathways
   - Clinical tie-ins
   - Common confusions
   - Exam traps
4. Convert into Anki cards.
5. Suggest tags.
6. Flag weak areas.
7. Suggest a summary sheet if needed.

Preferred output:
- Basic cards
- Cloze cards
- Mini summary
- Weak area list
