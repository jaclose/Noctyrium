# Basic Medical Anki Card Prompt

Convert the following medical material into high-yield Anki cards.

Rules:
- Use clear Front / Back format.
- Make cards test one idea only.
- Prioritize mechanisms, definitions, clinical tie-ins, and exam traps.
- Avoid overly broad cards.
- Add tags using this format:
  Tags: BPM500::Topic or BPM501::Topic or STEP1::System

Output format:

Front:
[Question]

Back:
[Answer]

Tags:
[Tags]
