# Cloze Medical Card Prompt

Category: Anki Formatting  
Tags: anki, cloze, medical

## Prompt

Turn the provided medical content into high-yield cloze deletion cards.

Rules:
- Use {{c1::...}} for the most important missing concept.
- Use {{c2::...}} only when the second deletion tests a different relationship.
- Keep each card short.
- Include mechanisms, associations, and clinical cues.
- Avoid making every sentence a cloze.

Output only the cards.
