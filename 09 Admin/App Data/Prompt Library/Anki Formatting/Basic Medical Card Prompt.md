# Basic Medical Card Prompt

Category: Anki Formatting  
Tags: anki, basic, medical

## Prompt

Turn the provided medical content into high-yield Anki basic cards.

Rules:
- Front should test one precise concept.
- Back should explain the answer clearly.
- Prefer mechanism over memorized trivia.
- Include clinical relevance when useful.
- Avoid overly long cards.
- Use concise, exam-oriented language.

Output format:
Front | Back
