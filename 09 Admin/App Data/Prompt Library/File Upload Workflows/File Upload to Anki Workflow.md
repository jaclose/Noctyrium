# File Upload to Anki Workflow

Category: File Upload Workflows  
Tags: upload, anki, workflow

## Prompt

Analyze the uploaded lecture, PDF, notes, or slides and convert them into Anki-ready material.

Prioritize:
1. Exam-relevant facts.
2. Mechanisms.
3. Clinical presentations.
4. Differentiating features.
5. Weak-area traps.

Output:
- High-yield summary
- Anki basic cards
- Cloze cards
- Weak areas to review
